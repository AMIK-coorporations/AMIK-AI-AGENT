# **App Name**: AMIK AI AGENT

## Core Features:

- Urdu Wake Word: Uses 'مصنوئی ذھانت' as wake word.
- Urdu Chat: Leverages the DeepSeek API tool to generate contextually appropriate Urdu responses based on user input.
- Urdu Voice Response: Uses ElevenLabs API for high-quality Urdu text-to-speech conversion, providing natural-sounding vocal responses.
- Live Data: Provides current information via the Brave Search API when requested by the user.
- Reactive Visuals: Implements animated elements synchronized to the voice input/output using audio levels.
- Complete Urdu support: UI in Urdu Language
- Dark Theme: Creates a sleek, terminal-inspired user interface with animated glows and cyberpunk aesthetics to emulate a dark hacking theme.

## Style Guidelines:

- Primary color: Electric Purple (#BE29EC) evokes the hi-tech hacking aesthetic and offers a vibrant base for the UI.
- Background color: Dark gray (#222222) provides contrast for other elements.
- Accent color: Electric Blue (#7DF9FF) provides a pop of color.
- Body and headline font: 'Space Grotesk' (sans-serif) for a techy, computerized aesthetic; longer text may use 'Inter' (sans-serif) for body
- Code font: 'Source Code Pro' (monospace) for the terminal-style display.
- Uses glyph-style icons related to terminal commands, network activity, or security protocols, all in Electric Blue to maintain the high-tech theme.
- Glow and pulse animations are employed throughout the interface to emphasize interactivity, data flow, and system processes, enhancing the 'live' and responsive feel of the dark hacking theme.
